# 公開用スキル方針

このリポジトリは、GitHub で共有できる AI エージェント向けスキルを管理するためのものです。Codex 全般の設定、ローカル環境設定、認証情報、キャッシュ、個人用 override はこのリポジトリでは管理しません。

## リポジトリ方針

- Git で管理するのは、公開してよいスキルとその付属資料だけに限定する
- 認証情報、セッション、キャッシュ、ローカルパス、端末依存設定はリポジトリに含めない
- このリポジトリは共有スキルの配布元として扱い、導入は `npx skills` を前提にする

## スキル共有方針

- 共有スキルは `skills/` 配下で管理する
- 各スキルは複数プロジェクトで再利用できる形を優先し、前提となる文脈がある場合は明記する
- スキル内に秘密情報、非公開 URL、端末固有の前提を埋め込まない
- 大きな万能スキル 1 つより、小さく合成しやすいスキルを優先する

## 作成ガイド

- 1 スキル 1 ディレクトリで構成する
- 入口ファイルは `SKILL.md` にする
- いつ使うか、どの手順を踏むか、何を避けるべきかを明記する
- 補助スクリプトや参照資料を含める場合も、公開可能な内容だけにする
- 例やサンプルは公開可能な内容だけにする

## 配布とローカル作業

- スキルのインストール方法は手動コピーや symlink ではなく `npx skills` を基本とする
- 依存インストールや agent ごとの設定は、利用者側のローカルセットアップで扱う
